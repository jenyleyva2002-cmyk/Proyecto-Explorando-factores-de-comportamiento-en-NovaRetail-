# Proyecto-Explorando-factores-de-comportamiento-en-NovaRetail-

Este proyecto es un análisis correlacional (exploratorio). Correlación ≠ causalidad.

El objetivo es generar un reporte de análisis de correlación claro, responsable y accionable, que permita entender qué comportamientos del cliente están más fuertemente asociados con el ingreso generado, sin caer en interpretaciones causales incorrectas.

NovaRetail+ es una plataforma de comercio electrónico en Latinoamérica con millones de usuarios. Para el cierre de este año 2024, el equipo de Crecimiento y Retención busca entender:

¿Qué factores del comportamiento del cliente están más fuertemente asociados con el ingreso anual generado?
